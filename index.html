Tic Tac Toe

It's a two player game in (3 X 3) grid one player has to mark '0' and other one 'X'. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

Snake Game

Snake is operated by L, R, U & D key. As snakes eats the insects size of snake and score both increases. On hitting boundries or snake himself leads to end of the game.

Scrambe Word

Player need to guess the right word. If guess is right player wins the game. If you need hints then hints are also availabe like starting and ending letter of the word. User can make infinite guesses untill right guess is not made.

Racing Car

Player needs to save his/her car from other cars moving in the same direction(lesser spped). User can use L & R key to move car left and right. Once your car hits any other car leads to game end.

Typing Test

Player can set a time for test like 30s, 60s etc. Then Paragraph will display user need to type words correct and faster. On the basis of correct words and speed typing speed of user is calculated in WPM(word per minute)

ASCII Code Detector

We often require ASCII Code for charcter while solving coding questions So,The instance you press any key on your keyboard ASCII Code for that character will display on screen.
